Stuff-filled update

This last week or so was pretty productive! I think the world ended a few times, the general state of everything everywhere got worse, and several of the things you always thought was good has apparently been for evil and malevolent causes. Welcome to humanity. (Some exaggeration implied.)

Locally, I made a few things! Chronological order is the easiest way to write, so..

I made a Dragon Quest IX alchemy thing. You give it the name of an alchemy recipe; it tells you all the ingredients that are needed, and all the ingredients required to make each of those ingredients, and so on (down to ingredients that can't be alchemized). That took a while to make - mostly because of the gigantic file storing all the recipes in the game (that I've discovered) that I typed out by hand. It mostly looks like this.

I worked on Ludum Depot 38 Cool Edition a bit. Ludum Depot 38 is a game eevee and glitchedpuppet made; Cool Edition is an endless (infinity waves) version of that game that I'm working on. Basically I implemented a few suggestions that Opa-Opa gave; read them if you're interested.

I spent an hour or so making a thing to download all the K'NEX manuals that are officially available online for the general public. Then I discovered that the directory containing all those PDFs is already easy to view. That did give me a bit of a reason to write a generic Apache-folder downloader, though! The output folder is currently sitting on my desktop and weighs well over 13 gigabytes. Those files are kind of pointless and should probably be removed from my hard drive.
